baseruby by coatl

MRI converted for use as a library and baseline interpreter

ruby (1.8.6 for now) without GPL code, and with a few other hacks for it that make it useful to me

Install

same as normal ruby

License

Ruby's license minus GPL/LGPL. Essentially, this means a permissive license, like the Artistic or MIT licenses. But the exact situation is complicated, as many different (permissive) licenses apply to different files. See the files COPYING and LEGAL for full details, but be aware that all GPL and LGPL code has been removed.
